# BioSure — AI Intelligence Networking for Healthcare

MIT Hackathon project.

A clinical intelligence platform for multiple myeloma. Surfaces commercial
opportunity (untapped patient flow at CAR-T capable HCOs) alongside clinical
decision support (CAR-T trial eligibility evaluation).

## Stack

- **Frontend:** React + TypeScript + Vite + Tailwind + shadcn/ui
- **Backend:** FastAPI (Python) + MongoDB Atlas (Motor async driver)

## Repo layout

```
frontend/   React app
backend/    FastAPI service
```

See `frontend/README.md` and `backend/README.md` for module-specific setup.
